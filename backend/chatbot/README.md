# Chatbot Banco - Backend

Aplicacion Spring Boot para demo de chatbot bancario con login, registro, catalogo de productos y flujo basico de ventas.

## Requisitos
- Java 21
- MySQL local

## Ejecucion
Desde esta carpeta (`backend/chatbot`):

```powershell
.\mvnw.cmd spring-boot:run
```

Aplicacion disponible en `http://localhost:8081`.

## Pruebas

```powershell
.\mvnw.cmd test
```
